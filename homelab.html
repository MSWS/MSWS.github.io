Homelab

Using Proxmox, each of these services is deployed using Docker. Base domain is managed by Cloudflare with proxying and caching where possible.



Services

Cloudflare

  • Cloudflare-DDNS

    Updates cloudflare's DNS entries to support dynamic IP changes.

  • Cloudflared

    Restricts SSH access to cloudflared tunnels.

Statistics

  • Grafana

    Visualizes data collected by prometheus.

  • Prometheus

    Routinely scrapes data from various services and stores it in a time-series database.

Swag

Responsible for certificate management and reverse-proxying via nginx.

NextCloud

Personal cloud drive for images, files, and friends.

Wakapi

Open-sourced wakatime website for coding statistics.

Code Server

Remote VS-Code instance for homework, LaTeX, and any other projects.

Uptime-Kuma

Public status page to easily monitor service uptime and status.
